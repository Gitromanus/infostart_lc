# Инфостарт Курс SM

Браузерное расширение для отслеживания курса и транзакций внутренней валюты $m на портале Infostart.ru.

## Описание

Расширение автоматически обновляет курс внутренней валюты $m с биржи Infostart, отображает статистику транзакций в рублях и $m, а также предоставляет удобный дашборд для мониторинга финансовых операций.

## Возможности

- **Автоматическое обновление курса**: Получение актуального курса $m с биржи Infostart каждые 3 секунды
- **Дашборд статистики**: Отображение сумм транзакций за сегодня, месяц и общий итог
- **Конвертация валют**: Показ значений как в $m, так и в рублях по текущему курсу
- **Переключение периодов**: Просмотр статистики за день, месяц или весь период
- **Сохранение данных**: Кэширование истории транзакций в локальном хранилище браузера

## Установка

1. Откройте браузер Google Chrome (или другой браузер на основе Chromium)
2. Перейдите в режим разработчика расширений: `chrome://extensions/`
3. Включите «Режим разработчика» в правом верхнем углу
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку с файлами расширения (`manifest.json`, `content.js`)

## Использование

### Обновление курса
- Перейдите на страницу биржи: `https://infostart.ru/stockexchange/`
- Расширение автоматически определит текущий курс и сохранит его

### Просмотр статистики
- Перейдите на страницу транзакций: `https://infostart.ru/transact/`
- Вверху страницы появится дашборд со статистикой
- Используйте переключатели для просмотра данных за разные периоды
