# Инфостарт Курс SM

Браузерное расширение для отслеживания курса и транзакций внутренней валюты $m на портале Infostart.ru.

## Описание

Расширение автоматически обновляет курс внутренней валюты $m с биржи Infostart, отображает статистику транзакций в рублях и $m, а также предоставляет удобный дашборд для мониторинга финансовых операций с AI-прогнозом дохода.

## Демонстрация

![Дашборд статистики транзакций](https://github.com/Gitromanus/infostart_lc/blob/main/dash.png?raw=true)

*На скриншоте показан дашборд со статистикой транзакций за сегодня, месяц и общий итог с конвертацией в рубли.*

## Возможности

- **Автоматическое обновление курса**: Получение актуального курса $m с биржи Infostart
- **Дашборд статистики**: Отображение сумм транзакций за сегодня, месяц и общий итог
- **Конвертация валют**: Показ значений как в $m, так и в рублях по текущему курсу
- **Переключение периодов**: Просмотр статистики за день, месяц или весь период
- **Сохранение данных**: Кэширование истории транзакций в локальном хранилище браузера
- **AI-прогноз дохода**: Интеллектуальный прогноз ожидаемого дохода до конца месяца на основе истории транзакций
- **Фоновые уведомления**: Уведомления об изменении курса $m и новых покупках даже когда страница Infostart не открыта (через `chrome.alarms`)
- **Догрузка истории**: Возможность загрузить историю транзакций за несколько страниц
- **Настраиваемые уведомления**: Выбор типов уведомлений и порога изменения курса (с поддержкой дробных значений)
- **Фильтрация операций**: Учитываются операции "Скачивание файла", "Платное скачивание файла" и "Начисление"
- **Исключение операций**: "Отмена лота" не учитывается в статистике
- **Скрытие прогноза**: Возможность отключить отображение AI-прогноза в настройках
- **📊 Статистика по публикациям**: Детальная таблица с разбивкой дохода по каждой публикации (решению) с сортировкой по доходу
- **📊 Stacked bar chart**: Детальный график дохода с разбивкой по решениям — каждый столбец показывает вклад каждой публикации
- **📋 Таблица решений**: Сортированная таблица всех публикаций с указанием $m, рублей, процентной доли и количества скачиваний

## Установка

1. Откройте браузер Google Chrome (или другой браузер на основе Chromium)
2. Перейдите в режим разработчика расширений: `chrome://extensions/`
3. Включите «Режим разработчика» в правом верхнем углу
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку с файлами расширения (`manifest.json`, `content.js`, `background.js`)

## Использование

### Обновление курса
- Расширение автоматически определит текущий курс и сохранит его
- Фоновый сервис проверяет курс каждые 30 минут

### Просмотр статистики
- Перейдите на страницу транзакций: `https://infostart.ru/profile/money/transact/`
- Вверху страницы появится дашборд со статистикой
- Используйте переключатели для просмотра данных за разные периоды
- Нажмите ⚙️ для открытия настроек

### Уведомления
- **Об изменении курса**: приходят при изменении курса $m более чем на заданный порог (по умолчанию 5%, поддерживаются дробные значения до 0.01%)
- **О покупках**: приходят при обнаружении новых транзакций скачивания или начисления
- Уведомления работают в фоне — даже когда вы не на сайте Infostart (Service Worker + `chrome.alarms`)
- Клик по уведомлению открывает страницу транзакций
- Уведомления автоматически закрываются через 10 секунд

### Настройки (⚙️)
- **История**: догрузка истории транзакций (количество страниц)
- **Уведомления**:
  - 🟢 Повышение курса $m (с порогом срабатывания)
  - 🔴 Понижение курса $m
  - 💰 Покупки
  - 🔔 Проверить уведомления (тест)
- **Отображение**:
  - Показывать/скрывать прогноз дохода
  - 📊 Детальный stacked bar chart (доход по решениям) — включает/отключает stacked bar chart
  - 📋 Таблица статистики по публикациям — включает/отключает таблицу с разбивкой по решениям
- **Цвет рублей**: синий цвет (`#1976d2`) для сумм в рублях в дашборде и таблице

### AI-прогноз дохода
- Автоматически рассчитывает ожидаемый доход до конца месяца
- Использует несколько методов прогнозирования:
  - Экстраполяция текущего месяца
  - Анализ аналогичных месяцев прошлых лет
  - Пропорциональное масштабирование
- Отображает достоверность прогноза (высокая/средняя/низкая)

### Статистика по публикациям
- Автоматически собирает данные по каждой публикации из описания транзакций
- Отображает таблицу со столбцами: #, Публикация, $m, ₽, % (процент от общего дохода), Кол-во
- Сортировка по убыванию дохода
- Цветовая индикация: каждое решение имеет свой цвет, соответствующий stacked bar chart
- Прогресс-бар процентной доли для наглядности

### Stacked bar chart
- Детальный график дохода, где каждый столбец разделён на сегменты по решениям
- Цветовая легенда под графиком с указанием суммарного дохода по каждому решению
- Включается/отключается в настройках (⚙️ → Отображение → Детальный stacked bar chart)

## Разрешения

- `storage` — для сохранения курса и истории транзакций в локальном хранилище браузера
- `notifications` — для показа системных уведомлений
- `alarms` — для периодической проверки курса и транзакций в фоне
- `https://infostart.ru/*` — доступ к страницам портала Infostart
